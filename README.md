## how to launch (windows):

in git bash:
```
git clone git@github.com:johnriderr/get-window-size.git
```
in cmd:
```
cd get-window-size
python -m venv venv
venv\Scripts\activate.bat
pip install -r requirements.txt
main.py
```
further to launch, only:
```
cd get-window-size
venv\Scripts\activate.bat
main.py
```
